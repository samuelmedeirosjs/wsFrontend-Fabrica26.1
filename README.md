# 📘 Rickpédia

Projeto desenvolvido como parte do processo seletivo da **Fábrica de Software (2026.1)**, com foco em construção de aplicações frontend modernas utilizando boas práticas de desenvolvimento, componentização e consumo de APIs.

---

## 🚀 Sobre o Projeto

Este projeto tem como objetivo aplicar, na prática, conceitos de desenvolvimento frontend utilizando tecnologias modernas do ecossistema JavaScript.

A aplicação foi construída com foco em:

- Estrutura escalável
- Componentização reutilizável
- Boas práticas com React / Next.js
- Consumo de APIs externas
- Organização de código profissional

---

## 🛠️ Tecnologias Utilizadas

- **React**
- **Next.js (App Router)**
- **TypeScript**
- **Tailwind CSS**
- **ESLint**
- **Prettier**

---

## 📂 Estrutura do Projeto

```bash
src/
│
├── app/            # Rotas e layout (App Router)
├── components/     # Componentes reutilizáveis
├── services/api       # Integração com APIs
├── types/          # Tipagens TypeScript
├── lib/          # Funções auxiliares
│
public/             # Arquivos estáticos
```

---

## ⚙️ Como Executar o Projeto

### 1. Clone o repositório

```bash
git clone https://github.com/samuelmedeirosjs/wsFrontend-Fabrica26.1.git
```

### 2. Acesse a pasta

```bash
cd wsFrontend-Fabrica26.1
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Execute o projeto

```bash
npm run dev
```

O projeto estará disponível em:

```bash
http://localhost:3000
```
